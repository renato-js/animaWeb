checkpointJS

A simple way to you call some event when the user is on a specific page coordinates.

-

-

What is it?

This is a component javascript (without Jquery library to make your project more lightweight) to you put some checkpoints on your page.

Some times we need to make some animations on our pages ONLY when user is seeing the element on screen. Now you can.

If you need to call some function only when your user is on footer, configure it with checkpointJS, is easy.

How to Use

  • Scroll down this page and look the headers coloring while you scroll the page.
  • Get the code - https://github.com/renato-js/checkpointJS.
